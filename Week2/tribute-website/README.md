# Tribute to Chuck Palahniuk 🖋️

A tribute website project created using only HTML and CSS. Dedicated to the life and works of **Chuck Palahniuk**, author of *Fight Club* and other groundbreaking novels.

## 🌐 Live Demo  
[View Live on GitHub Pages](https://burakteli.github.io/tribute-to-author)

## 🧠 Sections Included

- **Home** – Hero image and quote
- **Info** – Biography and career timeline
- **Books** – List of published works
- **Footer** – Page credit for assignment

## 💡 Technologies Used

- HTML5  
- CSS3  
- Responsive layout principles  
- Semantic tags and accessibility best practices
